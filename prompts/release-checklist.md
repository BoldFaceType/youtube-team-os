# Release Checklist

Run before every upload. Check all boxes before going live.

## Pre-Upload
- [ ] Final video rendered and reviewed (no audio drops, no jump cuts without intention)
- [ ] Thumbnail exported at 1280×720, file size < 2MB
- [ ] Title finalized (matches thumbnail promise, ≤60 characters preferred)
- [ ] Description written (includes chapters, links, tags)
- [ ] Tags added (10–15, mix of broad and specific)

## YouTube Studio
- [ ] Video uploaded to correct channel
- [ ] Thumbnail set
- [ ] Title set
- [ ] Description pasted and formatted
- [ ] Chapters confirmed in preview
- [ ] End screen elements placed (subscribe, recent video)
- [ ] Cards added (if applicable)
- [ ] Category set
- [ ] Not made for kids — confirmed
- [ ] Visibility set (Scheduled or Public)

## Post-Publish
- [ ] Community post drafted and scheduled
- [ ] Shorts/clips identified for repurposing
- [ ] Email blurb sent or scheduled
- [ ] 48-hour analytics check scheduled
- [ ] 30-day postmortem reminder set in state/notes/
